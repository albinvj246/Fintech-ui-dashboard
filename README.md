### [Live Demo 🚀](https://fintech-ui-dashboard.vercel.app/)

**Indel Money Dashboard** is a high-performance, enterprise-grade financial interface designed for gold loan auction management. Built with the latest web technologies, it empowers financial institutions to monitor overdue accounts, track recovery performance, and manage auction schedules with surgical precision.

---

## 📱 Fully Responsive
The dashboard is meticulously engineered to be **fully responsive**. Whether you are managing auctions from a high-resolution 4K monitor or tracking recovery on a mobile device, the interface adapts perfectly.
- **Desktop**: Features a persistent, togglable sidebar for deep-dive management.
- **Mobile/Tablet**: Transitions to a streamlined drawer-based navigation with a compact metrics grid for on-the-go monitoring.
- **Data Tables**: Intelligently handles complex financial ledgers with horizontal scrolling and fixed column alignment.

## 🚀 Tech Stack

- **Framework**: [Next.js 16.2.4](https://nextjs.org/) (App Router)
- **Library**: [React 19.2.4](https://react.dev/)
- **Styling**: [Tailwind CSS 4.0](https://tailwindcss.com/) (Next-gen styling engine)
- **Charts**: [Recharts 3.8.1](https://recharts.org/) (Dynamic recovery analytics)
- **Deployment**: [Vercel](https://vercel.com/)
- **Typography**: [Inter Variable](https://fonts.google.com/specimen/Inter)

## 🛠️ Key Features

- **Dynamic Navigation**: Collapsible sidebar with micro-animations and a responsive mobile drawer menu.
- **Real-time Metrics**: High-density cards for account classification (NPA, Auctionable, etc.).
- **Recovery Analytics**: Advanced bar charts showing monthly recovery trends against target goals.
- **Interactive Schedules**: Calendar-integrated auction tracking with status filters.
- **Market Data**: Live gold rate monitoring (22K & 24K) to inform auction valuations.
- **Financial Ledger**: A professional, data-dense table with perfect alignment for multi-metric financial records.

## 🏃 Getting Started

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/albinvj246/Fintech-ui-dashboard.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run the development server**:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) to view the dashboard locally.

## 📁 Project Structure
- `/app`: Next.js App Router structure.
- `/app/components`: Modular, reusable dashboard components.
- `/app/globals.css`: Tailwind v4 configuration and modern design tokens.
- `/public`: Static assets and icons.
